# Interactive Visualization of Spherical Harmonics
**Інтерактивна візуалізація сферичних гармонік**

### About the Project / Про проект
This is a personal research tool developed for the visualization of spherical harmonics and associated Legendre polynomials. It is designed to explore mathematical models and wave dynamics in a 3D environment.
(Цей проект — персональний дослідницький інструмент для візуалізації сферичних гармонік та асоційованих поліномів Лежандра.)

### Technical Details / Технічні деталі
* **Core**: Custom `math.js` library for orthonormal basis calculation.
* **Rendering**: Three.js (WebGL) with custom vertex normal calculations for smooth shading.
